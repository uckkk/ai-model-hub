# AI Model Hub

Discover Chinese AI models that are 10-100x cheaper than GPT-4, Claude, and other international models.

## Features

- 🌍 Multi-language support (English/Chinese)
- 💰 Real-time price comparison
- 🧮 Savings calculator
- 📚 Registration tutorials
- 🔌 API proxy service (coming soon)

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- next-intl (i18n)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Data Sources

- Hugging Face API
- LMSYS Chatbot Arena
- OpenRouter API
- Official provider documentation

## Deployment

Deploy to Vercel:

```bash
vercel
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your API keys.
